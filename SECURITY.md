# FreeBlocks Security Policy

If you find a security vulnerability, please visit the [security page](https://github.com/GreyKnight572/freeblocks/security) of the project repository to submit a [report](https://github.com/GreyKnight572/freeblocks/security/advisories/new).

For the full FreeBlocks Security Policy, please refer to the [Security Policy](https://docs.google.com/document/d/1EJkzs1pF6TGQ4NK6Y23In-HQavL7rdzplfwCPdI9Fos/edit?tab=t.0#heading=h.lru420wf6dfk) section of our project policies.
